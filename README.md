# MemeLibrary

A meme library app built with Expo, React Native, and TypeScript.

[![Open in Expo Go](https://img.shields.io/badge/Open%20in%20Expo%20Go-000020?logo=expo&logoColor=white)](https://expo.dev/preview/update?message=main&updateRuntimeVersion=1.0.0&createdAt=2026-03-21T13:46:15.000Z&slug=exp&projectId=0384d7bc-46fe-4f87-ba0f-9c97ef0cad38&group=168d8801-2bb7-4d84-a5a7-421a51e13484)

## Getting Started

```bash
npm install
npx expo start
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm start` | Start Expo dev server |
| `npm run lint` | Run ESLint |
| `npm run typecheck` | Run TypeScript type checking |
| `npm test` | Run tests |
| `npx prettier --check .` | Check formatting |
